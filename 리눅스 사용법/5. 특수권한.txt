[특수권한]

1. SETUID : 파일 실행 시 파일 소유자 권한 실행 
	- s or S로 표현 
	- 실행파일에만 적용 
	- 실행 파일이 실행되는 동안, 해당 파일의 소유자의 권한으로 변경 
	- octal : 4000번대, symbolic : u+s 

2. SETGID : 파일 실행 시 파일 소유그룹의 권한으로 실행
	- s or S로 표현 
	- 실행파일에만 적용 
	- 실행 파일이 실행되는 동안, 해당 파일의 소유그룹의 권한으로 변경 
	- octal : 2000번대, symbolic : g+s

3. Sticky Bit : 공용 디렉토리로 설정 
	- t or T로 표현 
	- 디렉토리에만 적용 
	- 누구나 자유롭게 사용 가능 
	- 파일의 삭제, 이름변경 등은 소유자와 관리자만 가능 
	- 게시판 
	- octal : 1000번대 , symbolic : o+t 

4. sudo : root의 권한을 잠시 빌려와서 명령어를 실행 
	$> sudo <명령어>

	</etc/sudoers 설정 파일>
	<유저명>ALL=(ALL)		ALL
